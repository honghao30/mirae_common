소스파일 기본
  • 소스파일의 이름은 알파벳 소문자, 밑줄( _ ) 으로만 작성한다.
  • 공동함수는 ui_common.js에 작성한다.
  • 소스파일의 확장자명은 .js 으로 작성한다.
  • 소스파일의 인코딩은 UTF-8 으로 작성한다.
  • 탭을 이용한 들여쓰기는 하지 않는다.
함수 안내
Function Description
handleDropdownMenu 커스텀 드랍다운 제어 .dropdown-menu__wrap
generateUniqueId UniqueId 아이디를 생성 해준다.
numComma 숫자에 콤마를 찍어준다.
checkTextArea form요소에 입력 테스트 카운터 한다. count, total클래스가 가진 엘리먼트 필수 존재해야 한다.
checkInputFocus input의 삭제 버튼 노출시키는 스크립트로 btn-remove 클래스를 가진 버튼이 존재해야 동작함.
bottomSheetHandle bottom sheet를 제어하는 함수, 닫기 버튼이 있는 경우는 type-modal 클래스를 붙인다.
tabMenus 탭메뉴 스크립트로 실행은 tabMenus('classname')으로 하며 index.js 에서 실행부분만 추가한다.
animateNumbers 숫자 애니메이션
openModal 모달창 열고 닫는 함수로,
트리거는 onclick="openModal(event)" modal-id="basic", target은 id="basic" 로 구성한다.
scroll_event 스크롤시 발생하는 모든 이벤트를 처리하는 함수,
기본으로 페이지 로드시 애니메이션 처리
displayAnimation 함수에서 추가적인 실행 함수 처리

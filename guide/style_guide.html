폰트(Font)
  • Nanum Square는 디지털 환경의 시각 매체에서 사용하는 미래에셋생명의 공식 서체로써, 미래에셋그룹의 일관된 아이덴티티 표현을 위하여 해당폰트를 기본으로 사용한다.
  • Nanum Square의 Extra Bold, Bold, Regular를 위계에 맞게 사용한다.
  • 폰트는 나눔스퀘어 : var(--fontMain)

Nanum Square extra bold, bold, regular

애니메이션(animation)
애니메이션 가이드이며 JS에서 실행 시점을 제어해야 한다. 
            .slidedown {animation:slidedown 1s ease-out both;}
            @keyframes slidedown {
                0% {opacity:0; transform:translateY(70px);}
                100% {opacity:1; transform:translateY(0px);}
            }
* 미리보기 상태의 코드는 예시이며 사용시 .css or .js 에 내용 참고 바람.
